SunnySideUp Holiday app README

Currently the holiday app is in two different codes, at the pre-combination stage.

This web application shows holiday recommendations, weather information, flight information and shows airports and bars in a map in another tab.

Enter to anaconda prompt:

pip install bokeh
pip install requests
pip install pandas
pip install  plotly
pip install amadeus
pip install ipywidgets

Double check the packages are upgraded first!! Enter to anaconda prompt:
pip install ipywidgets --upgrade
pip install bokeh --upgrade
pip install pandas --upgrade
